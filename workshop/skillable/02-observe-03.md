# Run Prompt 1

Send a simple, single-specialist request and watch the metrics light up.

1. [] In the playground, send this prompt:

   ++What flights are available from Chicago to Rome?++

1. [] Wait for the concierge to reply with flight options.
1. [] Open the **metrics** panel and review the evaluator scores for this turn.
1. [] Select **View trace** (or the trace icon) to open the trace, and note the
   **evaluations attached to the trace**.
